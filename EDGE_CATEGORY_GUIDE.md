# Edge Category Guide - Updated System

## Overview

The edge categories now properly combine **ownership buckets** with **leverage scores** to show where actual tournament edge exists.

## The Problem with Old System

**Old way:** Only looked at leverage score
- "Elite Leverage" could be a 45% owned player (no real edge!)
- Didn't distinguish between contrarian and chalk plays

**New way:** Combines ownership + leverage + value
- Shows ownership context (Mega Chalk, Chalk, Mid, Contrarian)
- Identifies where true differentiation exists
